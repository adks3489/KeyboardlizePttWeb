KeyboardlizePttWeb
==================

本套件為Chrome擴充套件，透過本套件可以讓你用鍵盤操作網頁板的PTT

目前支援按鍵：

##[文章列表畫面](https://www.ptt.cc/bbs/Gossiping/index.html)

###【基本命令】

閱讀文章      r →

###【移動瀏覽】

上篇文章      p k ↑

下篇文章      n j ↓

往前翻頁      ^B P PgUp

往後翻頁      ^F N PgDn

跳至首篇      Home 0

跳至末篇      End $

###【看板資料】

進入精華區    z

其它看板      s


##[文章閱讀畫面](https://www.ptt.cc/bbs/Gossiping/M.1408188026.A.085.html)

###【基本命令】

離開          q ←
